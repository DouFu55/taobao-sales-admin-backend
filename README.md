# 📊 财务报表与销售明细管理系统（后端）
本项目是配套前端系统的后端服务，基于 Golang + Gin 开发，采用 SQLite 轻量级文件数据库，内置前端打包文件，启动后端即可访问完整系统，无需额外部署，为中小电商商家提供轻量化的接口支撑，适配临时替代高成本商用ERP的使用场景。

## 🛠 技术栈
- 核心框架：Gin（高性能HTTP Web框架）
- 开发语言：Golang 1.20+
- 数据库：SQLite（轻量级文件数据库，无需独立部署）
- 静态文件处理：Go原生embed（内嵌前端打包文件）
- ORM框架：GORM（简化数据库操作）

## ✨ 核心功能
### 1. 销售明细管理接口
- Excel销售数据批量上传导入
- 销售明细列表查询、订单删除
- 订单发货、交易关闭、交易完成、退款金额处理

### 2. 产品成本管理接口
- 成本数据列表查询、新增、编辑、删除
- 支持SKU/商品名/供应商多维度搜索、成本自动计算

### 3. 推广支出管理接口
- 推广数据列表查询、新增、编辑、删除
- 按推广日期筛选、消耗金额统计

### 4. 其他支出管理接口
- 其他支出记录列表查询、新增、编辑、删除
- 按支出日期筛选、多类型支出分类管理

### 5. 财务报表分析接口
- 日报接口（指定日期）、自定义日期范围报表接口
- 月度报表接口（指定月份）、年度报表接口（指定年份）
- 毛利率、净利率等核心经营指标自动计算

### 6. 部署运行特性
- 前端内嵌部署：启动后端即可访问完整系统，无需单独部署前端
- 零额外依赖：无需安装数据库/WEB服务，单文件即可运行
- 一键启动：自动初始化数据库，启动后自动打开浏览器访问
- 优雅启停：监听系统信号，关闭时安全释放资源

## 🔗 前端适配
本后端服务为前端 Vue3 + TDesign 系统提供全量接口支持，内置前端打包文件，前后端一体化部署，无需跨域配置，本地启动即可使用完整系统。
- 前端仓库地址：https://github.com/DouFu55/taobao-sales-admin-frontend
